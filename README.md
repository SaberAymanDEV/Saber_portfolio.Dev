## 🎨 مميزات الموقع والبورتفوليو (Portfolio Core Features)
1. **المساعد الذكي (AI Assistant / Chatbot)**: chatbot مدمج يدعمه نموذج **Gemini 3.5 Flash** للإجابة على جميع تساؤلات الزوار حول خبراتي ومشاريعي وحجز الجلسات الاستشارية معي.
2. **لوحة تحكم المشرف (Admin CMS)**: لوحة تحكم سرية ومحمية بـ JWT بالكامل لإدارة المشاريع، مشاهدة الرسائل، تعديل السيرة الذاتية (CV)، وتأكيد أو إلغاء الحجوزات.
3. **نظام الحجز الآلي (Dynamic Table Bookings)**: يمكن للعملاء والزوار حجز مواعيد للاستشارات والاجتماعات البرمجية وتلقي تأكيدات آلية.
4. **حفظ البيانات المزدوج (Hybrid High-Durability DB)**: دعم تخزين كامل وإدارة مرنة من خلال قواعد بيانات **MongoDB** و **Google Firebase Firestore**، مع ميزة التخزين المحفوظ محليًا كدعم احتياطي مستمر.
5. **سرعة خارقة وتصميم متجاوب**: الموقع مجهز بتصميم عصري بالكامل باستخدام Tailwind CSS و Framer Motion لتوفير تجربة بصرية سلسة ومريحة في تصفح جميع الأجهزة والشاشات.

## 🛠️ التقنيات المستخدمة (Tech Stack)
* **Frontend**: React 18, Vite, Tailwind CSS, Framer Motion, Lucide Icons, Recharts (للإحصائيات).

## 🚀 التشغيل المحلي (How to Run Locally)

### المتطلبات الأساسية (Prerequisites):
- تثبيت بيئة عمل **Node.js** (إصدار 18 فما فوق).

### خطوات التشغيل:
1. قم بتهيئة ملف المتغيرات البيئية عبر نسخ `.env.example` إلى `.env` أو `.env.local`:
   ```bash
   cp .env.example .env
   ```
2. قم بإدخال بيانات الاتصال الخاصة بك في ملف `.env` (مثل مفتاح Gemini API ومستندات الاتصال بـ Firebase أو MongoDB):
   - `GEMINI_API_KEY`: مفتاح تشغيل الذكاء الاصطناعي للمساعد الشخصي.
   - متغيرات الـ Firebase أو MongoDB لتخزين بيانات الرسائل والمشاريع بنجاح.

3. قم بتثبيت مكاتب الاعتماد والأدوات:
   ```bash
   npm install
   ```

4. بدء تشغيل الخادم المحلي للتطوير:
   ```bash
   npm run dev
   ```
